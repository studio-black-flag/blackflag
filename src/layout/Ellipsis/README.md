# Container

Container servem pra demilitar uma largura máxima para o conteúdo.
## Marcação
```jsx
<Container className="bg-doc">
  Conteúdo
</Container>
```

## Variáveis SCSS
```scss
$container-max-widths: (
  sm: 540px,
  md: 720px,
  lg: 960px,
  xl: 1140px
) !default;
$container-padding-x: 2rem!default;
```
